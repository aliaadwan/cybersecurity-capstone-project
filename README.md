# Cybersecurity Capstone Project — NovaTrade Solutions Security Assessment

## Overview
This repository documents a comprehensive cybersecurity capstone project simulating a real-world security engagement for a mid-size e-commerce organization (NovaTrade Solutions). The project integrates three core cybersecurity disciplines: Vulnerability Assessment & Penetration Testing (VA/PT), Security Operations Center (SOC) incident response, and Governance, Risk & Compliance (GRC).

## Scenario
NovaTrade Solutions operates a hybrid technology environment including a customer-facing web portal, cloud-based email, remote VPN access, and internal file/database servers. Security monitoring detected indicators of compromise suggesting unauthorized access, triggering a full security assessment.

## Project Structure

| Folder | Description |
|--------|-------------|
| `01-VA-PT-Report/` | Vulnerability assessment & penetration testing findings, risk ratings, and remediation plan |
| `02-SOC-Incident-Response/` | Incident timeline, alert triage, containment actions, and recovery recommendations |
| `03-Risk-Register-Policy/` | Risk register and a focused security policy document (MFA Policy) |
| `presentation/` | Executive summary presentation of key findings and business impact |

## Skills Demonstrated
- Vulnerability identification and risk-based prioritization
- Incident timeline reconstruction and triage decision-making
- Security policy development aligned with NIST/ISO 27001
- Technical-to-business communication for executive stakeholders

## Tools & Frameworks Referenced
NIST CSF, ISO/IEC 27001, CVSS, MITRE ATT&CK
